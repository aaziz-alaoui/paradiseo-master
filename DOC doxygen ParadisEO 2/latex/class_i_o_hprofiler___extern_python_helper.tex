class I\+O\+Hprofiler\+\_\+\+Extern\+Python\+Helper$<$ Input\+Type $>$}

Helper class which call Python\textquotesingle{}s A\+PI without having to bother with types.

The public interface expose {\ttfamily get\+\_\+callable} to get a handle on an instance member function, and {\ttfamily call} to actually call it with some arguments.

The class details will take care of converting arguments and return types from/to Python\textquotesingle{}s data structures.
