All Individuals in order

The best individual first, then the others in sequence (random order). for G3 evolution engine, see Deb, Anad and Joshi, C\+EC 2002

As \mbox{\hyperlink{classeo_sequential_select}{eo\+Sequential\+Select}}, it is an \mbox{\hyperlink{classeo_select_one}{eo\+Select\+One}} to be used within the eo\+Easey\+EA algo, but conceptually it should be a global \mbox{\hyperlink{classeo_select}{eo\+Select}}, as it selects a bunch of guys in one go (done in the setup function now)
