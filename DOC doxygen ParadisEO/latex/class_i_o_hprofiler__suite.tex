\subsubsection*{template$<$class Input\+Type$>$\newline
class I\+O\+Hprofiler\+\_\+suite$<$ Input\+Type $>$}

A base class for construct a suite for sets of I\+O\+Hprofiler\+\_\+problems. 

To specify available problems of a suite, register\+Problem must be implemented in derived class. The default lable of problems are string type. Integer type are also optional, but we highly recommond registering problem with string lable and creating a map of string problem\+\_\+name and integer problem\+\_\+id.
