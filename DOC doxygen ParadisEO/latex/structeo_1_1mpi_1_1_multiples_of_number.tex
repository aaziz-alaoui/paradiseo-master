struct eo\+::mpi\+::\+Multiples\+Of\+Number$<$ E\+O\+T $>$}

Sends seeds to the workers, which are multiple of a number given by the master. If no number is given, a random one is used. 

This functor ensures that even if the same store is used with different jobs, the seeds will be different.
