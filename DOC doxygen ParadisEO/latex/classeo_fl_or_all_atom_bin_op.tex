\doxysubsection{Description détaillée}
\subsubsection*{template$<$class E\+OT$>$\newline
class eo\+Fl\+Or\+All\+Atom\+Bin\+Op$<$ E\+O\+T $>$}

Generic eo\+Bin\+Ops on fixed length genotypes. Contains exchange crossovers (1pt and uniform) and crossovers that applies an Atom crossover to all components with given rate, or to a fixed prescribed nb of components

Example\+: the standard bitstring 1-\/point and uniform crossovers could be implemented as resp. \mbox{\hyperlink{classeo_fl_or1pt_bin_op}{eo\+Fl\+Or1pt\+Bin\+Op}} and \mbox{\hyperlink{classeo_fl_or_uniform_bin_op}{eo\+Fl\+Or\+Uniform\+Bin\+Op}} Bin Crossover using an Atom Crossover that is applied to a A\+LL components with given rate
