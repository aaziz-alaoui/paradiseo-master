\doxysubsection{Description détaillée}
\subsubsection*{template$<$class T$>$\newline
class I\+O\+Hprofiler\+\_\+observer$<$ T $>$}

A class of methods of setting triggers recording evaluations. 

Four methods is introduced here\+:
\begin{DoxyEnumerate}
\item Recording evaluations by a static interval.
\item Recording complete evaluations.
\item Recording evaluations as the best found solution is updated.
\item Recording evaluations at pre-\/defined points or/and with a static number for each exponential bucket. 
\end{DoxyEnumerate}
