class I\+O\+Hprofiler\+\_\+\+Range$<$ R $>$}

Interface for classes that computes indices from ranges.

The idea is to input \mbox{[}min,max\mbox{]} and the discretization size and to compute the position in \mbox{[}0,size\mbox{]} of any value in \mbox{[}min,max\mbox{]}.
