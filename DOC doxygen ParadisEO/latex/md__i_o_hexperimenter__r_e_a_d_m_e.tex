{\bfseries{I\+O\+Hexperimenter}} {\itshape provides}\+:


\begin{DoxyItemize}
\item A framework for straightforward benchmarking of any iterative optimization heuristic
\item A suite consisting of 23 pre-\/made Pseudo-\/\+Boolean benchmarking function, with easily accessible methods for adding custom functions and suites
\item Logging methods to effortlesly store benchmarking data in a format compatible with {\bfseries{I\+O\+Hanalyzer}}, with future support for additional data logging options
\item ({\bfseries{Soon to come}}\+:) A framework which significantly simplifies algorithm design
