\subsubsection*{template$<$class Input\+Type$>$\newline
class I\+O\+Hprofiler\+\_\+problem$<$ Input\+Type $>$}

A base class for defining problems. 

Basic structure for I\+O\+H\+Experimentor, which is used for generating benchmark problems. To define a new problem, values of \textquotesingle{}number\+\_\+of\+\_\+variables\textquotesingle{}, \textquotesingle{}lowerbound\textquotesingle{}, \textquotesingle{}upperbound\textquotesingle{}, \textquotesingle{}best\+\_\+variables\textquotesingle{}, \textquotesingle{}optimal\textquotesingle{} must be given. An corresponding \textquotesingle{}internal\+\_\+evaluate\textquotesingle{} method must be defined as well.

Basic structure for I\+O\+H\+Experimentor, which is used for generating benchmark problems. To define a new problem, the \textquotesingle{}internal\+\_\+evaluate\textquotesingle{} method must be defined. The problem sets as maximization by default. If the \textquotesingle{}best\+\_\+variables\textquotesingle{} are given, the optimal of the problem will be calculated with the \textquotesingle{}best\+\_\+variables\textquotesingle{}; or you can set the optimal by defining the \textquotesingle{}customized\+\_\+optimal\textquotesingle{} function; otherwise, the optimal is set as min()(max()) for \mbox{\hyperlink{structmaximization}{maximization(minimization)}}. If additional calculation is needed by \textquotesingle{}internal\+\_\+evaluate\textquotesingle{}, you can configure it in \textquotesingle{}prepare\+\_\+problem()\textquotesingle{}.
