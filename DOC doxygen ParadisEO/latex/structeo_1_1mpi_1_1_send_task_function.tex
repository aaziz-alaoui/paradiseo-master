struct eo\+::mpi\+::\+Send\+Task\+Function$<$ Job\+Data $>$}

Functor (master side) used to send a task to the worker. 

The user doesn\textquotesingle{}t have to know which worker will receive a task, so we just indicate to master the rank of the worker. The data used for computation have to be explicitly sent by the master to the worker, with indicated rank. Once this functor has been called, the worker is considered busy until it sends a return message to the master.

This is a functor implementing void \mbox{\hyperlink{classeo_u_f_a786e028409366dc273e19104f17ba68a}{operator()(int)}}, and also a shared data function, containing wrapper on its own type.
